MeshCore firmware for 

MeshCore v1.16 firmware compiled for the Waveshare RP2040-LoRa module (RP2040 + SX1262), available in four variants:
Companion Radio USB — connects to a laptop or desktop via USB serial, operated through app.meshcore.nz (Chrome/Edge). 
Best suited for fixed or mobile stations paired with a computer or phone..
Repeater — standalone mesh node that forwards messages across the network with no user interface. Ideal for fixed installations such as hilltops, towers, or buildings.
Room Server — group channel server that manages shared messages for a password-protected channel. Suitable for local groups or node clusters.
Terminal Chat — text serial interface for direct communication without a mobile app.
Common radio config: 869.618 MHz · BW 62.5 kHz · SF8 · 22 dBm · EU_868 MeshCore
Built and tested by YO2LDK for the YO/YU  LoRa MeshCore network.
